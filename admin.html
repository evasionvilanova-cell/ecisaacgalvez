Connecta Google Sheets

Configura una sola vegada. Segueix els 4 passos.

Pas 1 — Crea el Google Sheet

Ves a sheets.google.com, crea un full nou i posa-li el nom ecisaacgalvez

Pas 2 — Apps Script

Al Sheet: Extensions → Apps Script → esborra tot → enganxa el fitxer Code.gs → guarda

Pas 3 — Desplega

Deploy → New deployment → Web App → Execute as: Me → Access: Anyone → Deploy → autoritza → copia la URL

Pas 4 — Enganxa la URL

No s'ha pogut connectar. Comprova la URL i els permisos.
